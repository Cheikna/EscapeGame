# EscapeGame
Moteur de jeu utilisé : Unreal Engine 4
Langage utilisé : C++

Règles du jeu :
- Vous devez sortir de la pièce.
- Pour sortir de la pièce vous devez placer des éléments à un endroit spécifique de la pièce.
- Afin de prendre un objet, maintenez le clic droit de la souris ou maintenez Shift.
